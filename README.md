# NetC
A script to easily install a bootc image to a disk
==================================================
# Requirements:
Fedora host
`simple-term-menu` | Insall with `pip install simple-term-menu` to install pip run `dnf install python3-pip`
`nmtui`
`fastfetch`

# Installation
Copy `main.py` to `/usr/share/bootcrew/netc/`
